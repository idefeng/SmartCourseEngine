#### 2.2 API服务 (`src/services/api.ts`)
```typescript
// 请求拦截器 - 自动添加Token
apiClient.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

// 响应拦截器 - 统一错误处理
apiClient.interceptors.response.use(
  response => response.data.data, // 提取data字段
  error => {
    // 统一错误处理逻辑
    if (error.response?.data?.error_code === '0004') {
      window.location.href = '/login'
    }
    return Promise.reject(error)
  }
)
```

## 安全特性

### 1. 密码安全
- **bcrypt哈希**：12轮加密
- **密码强度**：至少8位，包含字母、数字、特殊字符
- **盐值生成**：每个密码使用唯一盐值

### 2. 令牌安全
- **JWT签名**：HS256算法
- **短期令牌**：访问令牌24小时过期
- **刷新机制**：使用长期刷新令牌
- **Token类型**：区分访问令牌和刷新令牌

### 3. 输入验证
- **请求验证**：Pydantic模型验证
- **SQL注入防护**：参数化查询
- **XSS防护**：输出编码
- **CSRF防护**：SameSite Cookie
