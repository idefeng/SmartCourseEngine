## 技术架构

### 完整的7个微服务架构
```
用户界面层 (前端/API客户端)
        ↓
    API网关层 (FastAPI, 端口: 8000)
        ↓
   业务逻辑层 (7个微服务)
   ├── 课件生成服务 (端口: 8001)
   ├── 视频分析服务 (端口: 8002)
   ├── 知识提取服务 (端口: 8003)
   ├── 搜索服务 (端口: 8004)
   ├── 推荐服务 (端口: 8005)
   └── 通知服务 (端口: 8006)
        ↓
   数据处理层 (PostgreSQL + Neo4j + Redis)
        ↓
   AI模型层 (Whisper + NLP + 推荐算法)
```

### 技术栈详情

#### 后端技术栈
- **框架**: FastAPI + Pydantic + SQLAlchemy
- **数据库**: PostgreSQL (主数据) + Neo4j (知识图谱) + Redis (缓存)
- **消息队列**: RabbitMQ (异步任务)
- **文件存储**: MinIO (对象存储)
- **监控**: Prometheus + Grafana
- **容器化**: Docker + Docker Compose

#### 微服务通信
- **HTTP/REST**: 同步API调用
- **异步客户端**: httpx异步HTTP客户端
- **服务发现**: Docker Compose网络
- **健康检查**: 每个服务都有/health端点
