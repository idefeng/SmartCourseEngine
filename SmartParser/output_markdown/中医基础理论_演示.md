# 中医基础理论：阴阳五行学说（演示版）

## 一、阴阳学说
阴阳学说是中国古代哲学的一对范畴，是对自然界相互关联的某些事物和现象对立双方属性的概括。

### 1.1 阴阳的基本概念
- **阳**：凡是向光的、外向的、上升的、温热的、明亮的、活跃的属性。
- **阴**：凡是背光的、内守的、下降的、寒冷的、晦暗的、静止的属性。

### 1.2 阴阳的相互关系
- **对立制约**：阴阳双方在一定限度内动态地对立制约，维持平衡。
- **互根互用**：阴阳双方互为存在的前提。
- **消长平衡**：阴阳双方不是静止的，而是处在不断的运动变化中。
- **相互转化**：在一定条件下，阴阳可以向其对立面转化。

## 二、五行学说
五行，即木、火、土、金、水五种物质及其运动变化。

### 2.1 五行的特性
- **木**：木曰曲直。引申为生长、升发、条达、舒畅。
- **火**：火曰炎上。引申为温热、上升、明亮。
- **土** : 土爰稼穑。引申为生化、承载、受纳。
- **金** : 金曰从革。引申为沉降、肃杀、收敛。
- **水** : 水曰润下。引申为寒凉、滋润、向下。

### 2.2 五行的生克乘侮
- **相生**：木生火，火生土，土生金，金生水，水生木。
- **相克**：木克土，土克水，水克火，火克金，金克木。

## 三、临床应用
阴阳五行学说贯穿于中医学的生理、病理、诊断和治疗等各个方面。
- **病理**：阴阳失调是疾病的基本病机。
- **诊断**：察颜悦色，辨别阴阳。
- **治疗**：调整阴阳，以平为期。
