## 🏗️ 新架构亮点

### 1. **五层微服务架构**
```
用户界面层 → API网关层 → 业务逻辑层 → 数据处理层 → AI模型服务层
```

### 2. **7个微服务**
1. **api-gateway** (端口: 8000) - API网关和路由
2. **course-generator** (端口: 8001) - 课件生成（保留原有功能）
3. **video-analyzer** (端口: 8002) - 视频分析（新增）
4. **knowledge-base** (端口: 8003) - 知识库管理（升级）
5. **search-engine** (端口: 8004) - 智能检索（新增）
6. **recommendation** (端口: 8005) - 推荐服务（新增）
7. **notification** (端口: 8006) - 通知服务（新增）

### 3. **多数据库支持**
- **PostgreSQL** - 结构化数据存储
- **Neo4j** - 知识图谱关系存储
- **Pinecone** - 向量相似性搜索
- **Redis** - 缓存和会话管理
- **MinIO** - 对象存储（视频/文档）

### 4. **AI模型集成**
- **Whisper** - 语音识别和转录
- **CLIP/YOLO** - 视觉内容分析
- **BERT/LLM** - 文本理解和知识提取
- **GPT-4V** - 多模态内容理解
