### 5. 开发环境配置 ✅

#### 5.1 快速启动
```bash
# 1. 进入项目目录
cd SmartCourseEngine/new_architecture/admin-frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问地址
# http://localhost:3000
```

#### 5.2 环境变量
```env
# .env.local
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

#### 5.3 构建部署
```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm run start

# Docker部署
docker build -t smartcourse-admin .
docker run -p 3000:3000 smartcourse-admin
```
